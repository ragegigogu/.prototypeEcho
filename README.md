# .prototypeEcho
Una interesante travesia en el mundo de las micro ondas

Prototipo para la detección de objetos estáticos o en movimiento

Que mediante el uso de un sensor tipo sonico mapeara zonas de rango extendido para clasificar objetos y determinar su posicion en el tiempo y espacio; con el fin de concluir si dicho objeto esta inmovil o experimenta algun tipo de aceleracion y su direccion.
